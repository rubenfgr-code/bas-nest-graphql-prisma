# Base project

- **[NestJS](https://nestjs.com/)**
- **[GraphQL](https://graphql.org/) over [Apollo GrapHQL](https://www.apollographql.com/)**
- **[Prisma](https://www.prisma.io/)**

## Installation

```bash
$ npm install
$ npx prisma migrate dev
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
